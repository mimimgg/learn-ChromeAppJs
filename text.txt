* Boolean Type
  : true(참)과 false(거짓)만을 반환한다
  
  - 사용예시
    1) 사용자가 로그인을 했는가?
    2) 웹사이트가 로딩되고 있는가?
  
  - 주의사항
    null : 변수에 아무것도 없는 상태
    (null은 절대 자연적으로 발생하지 않음)
    undefined : 변수는 선언했지만 값을 할당하지는 않음

* Array
  : 데이터를 정리함

  - 사용예시
    1) 한 주간의 요일 만들기 
       [mon, tue, wed, thu, fri, sat, sun]

  - 주의사항
    1) string을 구분한다
    2) number를 구분한다.

* Push
  : 무언가를 추가함
